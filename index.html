Plexus Archiver

Collection of Plexus components to create archives or extract archives to a directory with a unified Archiver/UnArchiver API whatever the archive format is.

Interface Components for supported formats (as Plexus role hint)
Archiver bzip2, xz, dir, ear, gzip, jar, mjar, snappy, tar, war, zip. Notice that rar component produces a jar
UnArchiver bzip2, xz, gzip, snappy, tar, zip (also available as jar, war, ear, swc, nar, esb, sar, car, par, rar), tgz or tar.gz, tbz2 or tar.bz2, txz or tar.xz, tar.snappy.
