Plexus Archiver

Collection of Plexus components to create archives or extract archives to a directory with a unified Archiver/UnArchiver API whatever the archive format is.

Interface Components for supported formats (as Plexus role hint)
Archiver bzip2, dir, ear, gzip, jar, mjar, rar (notice produces a jar), snappy, tar, tar.bz2 (tbz2), tar.gz (tgz), tar.snappy, tar.zst (tzst), tar.xz (txz), war, xz, zip, zst
UnArchiver bzip2, gzip, snappy, tar, tar.bz2 (tbz2), tar.gz (tgz), tar.snappy tar.zst (tzst), tar.xz (txz), xz, zip (also available as car, ear, esb, jar, nar, par, rar, sar, swc, war), zst
